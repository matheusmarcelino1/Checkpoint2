# Sistema de Processamento de Pagamentos

## Descrição

Este projeto consiste em uma aplicação console desenvolvida em C# com o objetivo de simular um sistema de processamento de pagamentos.

O sistema permite ao usuário escolher entre pagamento via **Cartão** ou **Boleto**, inserir os dados necessários e visualizar um resumo da operação realizada.

* Menu interativo no console

* Loop contínuo até o usuário optar por sair
